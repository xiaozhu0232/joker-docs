    操作命令¶

    传送地图功能¶

    功能：将玩家传送到指定地图坐标

    格式： 

    MOVE 地图编号 坐标X 坐标Y 地图区域 ;地图编号查看说明书上的地图编号表,地图区域查看地图区域表

    例： 

    #IF 
#ACT 
MOVE 143 1107 607 0 
LocalMessage "你传送到了比奇省(1107,607)" 
FALSE

    传送地图玩家¶

    功能：将指定地图的所有玩家传送到另一个地图

    格式： 

    MoveAll 原地图id 目标地图id 区域类型 (区域类型请查看说明书上的地图区域类型表)

    例： 

    #IF 
#ACT 
MoveAll 147 143 0 
TopMessage "已将147(土城)的所有玩家传送到143(比奇省)" 
TRUE 
Break

    功能：修改玩家性别

    例： 

    #IF 
#ACT 
CHANGEGENDER 
男性 
Break

    --------------------------------------------------------------------------------------------------------------------- #IF

    ACT¶

    CHANGEGENDER 女性 Break

    你变成了一个美女!

    修改玩家职业 格式 ：CHANGECLASS 职业类型 (战士,法师,刺客,弓手, 道士,龙枪)

    例： 

    #IF 
#ACT 
CHANGECLASS

Break 
#SAY 
你的职业变成了一个战士!

    提示：转职或变性需要小退后生效！

    修改物品属性¶

    功能：修改调整身上指定装备属性

    格式： 

    ChangeItemValue 0 装备位置 装备属性 属性点数

    例： 

    #IF 
#ACT 
ChangeItemValue 0 0 2 50 
ChangeItemValue 
0 0 1 50 ;修改装备升级次数,否则不显示

**提示：**装备位置可查看说明书的装备位置表，装备属性可查看装备属性表

    装备属性增加示例： 

    #IF 
#ACT 
MOV P1 <$ItemValue(0,0,2)> 
;将身上武器的攻击属性存入变量P1

;设置武器攻击属性为P1变量的值

"你的武器攻击属性提升1点" FALSE

    发放扣除物品¶

    功能：发放和扣除玩家背包物品相关

    发放物品格式1： GiveItem 物品名称 数量 是否绑定(TRUE为绑定、FALSE为不绑定)

    例： 

    #IF 
#ACT 
GiveItem  屠龙 1 
FALSE 
#SAY 
你获得了1把未绑定的屠龙!

    发放物品格式2： GiveItemIdx 物品编号 数量 是否绑定(TRUE为绑定、FALSE为不绑定)

    提示：物品编号请查看物品代码表或版本内物品数据

    例： 

    #IF 
#ACT 
GiveItemIdx 99900026 1 
TRUE 
Break 
#SAY 
你获得了1把绑定的99900026!

    扣除物品格式1： TakeItem 物品名称 数量

    例： 

    #IF 
#ACT 
TakeItem 屠龙 
1 
#SAY 
你失去了1把屠龙!

    扣除物品格式2： TakeItemIdx 物品编号 数量

    例： 

    #IF 
#ACT 
TakeItemIdx 99900026 
1 
#SAY

    发送消息设置¶

    功能：游戏发送提示公告

    全服公告提示格式： 

    GlobalMessage 发送内容
    (是否滚动 FALSE or TRUE) ;FALSE 代表固定提示 TRUE 代表滚动提示

    例： 

    #IF 
#ACT 
GlobalMessage "发送全服提示的固定消息" 
FALSE

    系统公告提示格式： 

    LocalMessage 发送内容
    (是否全服 FALSE or TRUE) ;FALSE 代表个人提示 TRUE 代表全服提示

    例： 

    #IF 
#ACT 
LocalMessage "发送一条全服能看到的消息" 
TRUE

    顶部公告提示格式： 

    TopMessage 发送内容
    (是否全服 FALSE or TRUE) ;FALSE 代表个人提示 TRUE 代表全服提示

    例： 

    #IF 
#ACT 
TopMessage "发送一条全服能看到的消息" TRUE

    修改公告字体大小格式： 

    <font size='25'>[文本内容]</font>

    例： 

    #IF 
#ACT 
LocalMessage 
"<font size='25'>[你是一条小丑鱼]</font>！" TRUE< 
/FONT> ;提示: 
加颜色代码的消息有空格的地方请使用大空格(按TAB键) < /FONT>

    修改消息提示颜色格式： 

    <font
    color='十六进制颜色值'>[文本内容]

    例： 

    #IF 
#ACT 
LocalMessage 
"<font color='#6AE6FC'>[你全家都是小丑鱼]</font>！" TRUE ;提示: 
加颜色代码的消息有空格的地方请使用大空格(按TAB键) < /FONT> ---------------------------------------------------------------------------------------------------------------------

    地图上放物品¶

    功能：在指定地图坐标上放物品或装备

    格式： 

    THROWITEM 地图名称 X坐标 Y坐标 范围 物品名称 数量 时间(秒) 是否掉落提示(0不提示,1提示) 是否可以立即捡取(0不能立即捡取,1可以立即捡取) 叠加物品是否叠加(0或空叠加,1非叠加)
    触发几率(为空或等于1时百分之百触发) (0或空所有人,1仅自己或队伍拾取，如果设置了这个,前面的立即拾取和限定时间失效)

    例： 

    #IF 
#ACT 
THROWITEM 盟重省 940 523 1 裁决之杖 1 60 1 0 1 
LocalMessage 
“盟重省()940,523"附近出现了一把裁决之杖 TRUE

    IF¶

    ACT¶

    THROWITEM 盟重省 940 523 1 裁决之杖 1 60 1 0 1 50 ;1/50的几率触发

    AddList 字符 文本路径

    例：

Break 
#SAY

    删除文本指定字符格式： 

    DelList 字符 文本路径

DelList <$USERNAME> 
..\会员列表\会员名单.txt 
Break 
#SAY 
已删除你的会员特权！ 
#ELSESAY 
你目前还不在会员名单内呢！
