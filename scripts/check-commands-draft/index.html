    检测命令¶

    检测人物在线¶

    功能：检测指定人物是否在线

    格式： 

    CheckOnline 人物名

    例： 

    #IF 
CheckOnline 猴哥 
#SAY 
玩家猴哥在线！ 
#ELSESAY 
玩家猴哥不在线,不知道干什么去了！

    检测人物称号¶

    功能：检测人物是否拥有或佩戴指定称号

    提示：称号编号请查看说明书上的称号数据表

    检测是否拥有指定称号格式： 

    CheckTitle 称号编号

    例： 

    #IF 
CheckTitle 
14 
#ACT 
Break

#ELSESAY 
你当前角色没有"新手上路"称号！

    检测指定角色是否拥有指定称号格式： 

    CheckPlayerTitle 角色名 称号编号

    IF¶

    Break

    ELSESAY¶

    角色张三没有"新手上路"称号！

    检测当前是否佩戴指定称号格式：

    检测元宝数量¶

    功能：检测玩家元宝数量

    格式： 

    CHECKGameGold 检测符 检测数值

    检测符支持: > ; 大于 < ; 小于 >= ;大于或等于 <= ;小于或等于 == ;等于 !=  ;不等于

    提示：由于传奇永恒元宝个位以下还有两位数，检测时得多出个位以下的两位数字(比如检测元宝是否大于100 就得 检测是否大于10000)

    例： 

    #IF 
CHECKGameGold  > 
10000 
#ACT 
Break 
#SAY 
身上元宝大于100个!

    IF¶

    CHECKGameGold  >= 10000

    ACT¶

    Break

    SAY¶

    身上元宝大于或等于100个!

    检测命令取反¶

    功能：反向检测判断

    格式： 

    NOT 检测命令 检测符
    检测条件

    检测符支持: > ; 大于 < ; 小于 >= ;大于或等于 <=
    ;小于或等于 == ;等于 !=  ;不等于

    例： 

    #IF 
NOT 
CHECKGameGold  > 10000 
#ACT 
Break 
#SAY 
身上元宝不大于100个!

    检测地图怪物数量¶

    功能：检测地图是否刷新指定怪物

    格式： 

    CheckMapSameMonCount 地图名称 怪物名称 控制符(<,>,=
    ) 数量是否忽略怪物名称后面的数字(0=检测数字,1=不检测数字,可以省略)

    例： 

    测试怪物名字不带数字写法 

#IF 
CheckMapSameMonCount 143 稻草人 > 10 
1 
#SAY 
比奇省的怪物“稻草人" 大于10只！

    测试怪物名字带数字写法

    IF¶

    CheckMapSameMonCount 143 稻草人1 > 10 0

    SAY¶

    比奇省的怪物 "稻草人1” 大于10只！
