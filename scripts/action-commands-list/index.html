    BUFF触发脚本¶

    功能：BUFF触发 00-QFunction.txt 脚本

    添加BUFF时触发： 在 00-QFunction.txt 文件中 添加 [@_ buff _add(< BUFF 编号>)]

    例： 

    [@_buff_add(4501)] 
#IF 
#ACT 
TopMessage  "你成功添加了编号为4501的BUFF" 
FALSE

    删除BUFF时触发： 在 00-QFunction.txt 文件中 添加 [@_ buff _delete(< BUFF 编号>)]

    例： 

    [@_buff_delete(4501)] 
#IF 
#ACT 
TopMessage  "你成功删除了编号为4501的BUFF" 
FALSE

    移除BUFF时触发： 在 00-QFunction.txt 文件中 添加 [@_ buff _remove(< BUFF 编号>)]

    例： 

    [@_buff_remove(4501)] 
#IF 
#ACT 
TopMessage  
"你成功移除了编号为4501的BUFF" FALSE

    执行BUFF时触发： 在 00-QFunction.txt 文件中 添加 [@_ buff _run(< BUFF 编号>)] [@_buff_run(4501)]

    IF¶

    ACT¶

    TopMessage  "编号为4501的BUFF成功执行1次，执行时间按BUFF间隔时间计算" FALSE

    IF¶

    ACT¶

    THROWITEM 盟重省 940 523 1 裁决之杖 1 60 1 0 1 50 ;1/50的几率触发
