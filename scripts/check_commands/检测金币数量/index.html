    检测金币数量¶

    功能：检测角色金币数量

    格式： 

    CHECKGOLD 检测符 检测数值

    检测符支持: > ; 大于 < ; 小于 >= ;大于或等于 <= ;小于或等于 == ;等于 !=  ;不等于

    例： 

    #IF 
CHECKGOLD < 
5000 
#ACT 
Break 
#SAY 
身上金币不足[5000]!

    IF¶

    CHECKGOLD >= 5000

    ACT¶

    Break

    SAY¶

    身上金币大于或等于[5000]!
